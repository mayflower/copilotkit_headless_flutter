# Code of Conduct

This project is intended to be a practical, respectful place to build agent
frontend infrastructure for Flutter.

## Expected Behavior

- Be direct and kind.
- Assume good intent while still asking for precise technical evidence.
- Keep discussions focused on the code, API behavior, documentation, and user
  outcomes.
- Make space for contributors with different experience levels.
- Disclose conflicts of interest when they materially affect a proposal.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private information without explicit permission.
- Repeatedly derailing technical discussions after maintainers ask to refocus.
- Using project spaces for spam, sales outreach, or unrelated promotion.

## Enforcement

Maintainers may edit, hide, or delete comments; close issues or pull requests;
or restrict participation when behavior harms the project. Enforcement should be
proportional, documented where appropriate, and focused on keeping collaboration
healthy.

To report a concern, contact the maintainers privately through GitHub. Security
vulnerabilities should follow `SECURITY.md`.
